Help Page

The below sections will describe the processes involved for ensuring that your word file is ready to be numbered and the steps involved in numbering the document.

Save file as .docx

In order for this application to function correctly, you must ensure that your document is in the docx format. To do this, click on File - Save As and then from the dropdown list choose the .docx format for Microsoft (This is the default for Microsoft Word) Then click save.

Add Headings to word document

For the purpose of deciding where to start page numbering in your document Heading 1 headings are required. To add a Heading 1 heading go to the Home tab, and in the style section click on Heading 1, then type in the name of your heading. Once your headings are added save your file as a .docx file.

Numbering word document on Numberingat.com

On the homepage, click on the Upload Word File button. This should bring up an explorer, navigate to your chosen file. Select the file and click Open.

Select Heading for numbering page

This will bring you to the Settings page. On the left-hand side, it will show the documents headings. Choose which heading will start the page numbering from. Once selected, the paragraph number corresponding to the selected heading will be shown in the top text box on the right. Please note that if your chosen heading is the first paragraph in the document then the document will only have one section.

Select numbering position

The page position drop down denotes where at the bottom of the page the numbers will be positioned, by default it will be positioned to the left.

Select numbering style

The final drop-down list lets you select the page numbering style for the first section of the document (often journal style documents, will have Roman numerals). The main section of the document will be numbered using ordinal numbers (1, 2, 3...).

Once you are happy with your settings click on the Set Page Number Button, the refresh button will return the user to the home page.

Download numbered file

After the document has been processed, the user will be brought to the download page. Here they can download the numbered Word file. If they do not download the file within two minutes, it will be automatically deleted. Please note that all files will be deleted to ensure space on the server.
