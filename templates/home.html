Page Numbers at X

To allow this software to number your page starting from a specific page please follow the following instructions

  • Upload a word document which contains headings.
  • Select one of the headings that is returned by clicking on it, which will be the heading for the starting page.
  • Select the position of the numbers on the page, default is left
  • Select the number format for the first section of the document
  • Click the Set Page Number button
  • Download your numbered file after it has been processed. Please not that all files will be deleted after 2 minutes.
